Workday Helper Options

Options

Configure your Workday connection and extension behavior.

Workday Connection

The base URL you use to access Workday. Paste everything up to and including your company/tenant name. Used as a fallback if the Time page URL is not set.

Direct link to your weekly time entry view. Navigate to Enter My Time and use the Set as Quick Link button on the helper panel to save it automatically. Or paste the URL manually here. Clicking the extension icon will open this page directly.

Settings saved!
Workday Schedule (Mon–Fri)

Two blocks are added per workday — morning and evening around your work hours.

Block 1 — Morning

Block 2 — Evening

Weekend Schedule (Sat–Sun)

One block covers the entire weekend day.

How to Use
Extension icon Click to open your Workday time entry page directly.
Right-click icon Choose Open Workday Time from the context menu.
Time landing page The panel shows Enter This Week to jump into the weekly view, and Set as Quick Link to save the URL so the extension icon opens it directly next time.
Enter Time / Quick Add The panel shows Workdays, Weekend, and Clear Empty Rows to automate on-call time entry.
After automation Review the filled entries, then click OK manually to submit.
